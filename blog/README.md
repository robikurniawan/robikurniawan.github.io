# Cara Menulis Blog (Gampang!)

## 2 Langkah Saja

### 1. Buat file `.md`
Buat file di folder `blog/` dengan nama: `judul-post.md`

Di awal file tambahkan:
```yaml
---
title: Judul Post
date: 2025-03-18
---
```
Lalu tulis konten dengan Markdown.

### 2. Tambah satu baris di list.txt
Buka `blog/list.txt` dan tambahkan nama file (tanpa .md):
```
hello-world
judul-post
```

Selesai! Push ke GitHub.
